FinWise Academy

Apr 2024 - May 2024

FinWise Academy is a web-based platform focused on enhancing financial literacy and helping users achieve financial independence. It offers a suite of financial calculators (SIP, FD, Loan EMI, PPF, EPF, CAGR, Rental Yield, Net Salary, Inflation) to support budgeting, saving, and investment planning. The platform features a personalized dashboard where users can access calculators, view/download their calculation history (TXT/Excel), and manage their accounts. Built with HTML, CSS (Bootstrap), JavaScript, and PHP for dynamic interactions, it includes smooth modals, animations, and AJAX-based login/signup functionality. User data is stored securely in a MySQL database (user_auth), with prepared statements and SSL deployment practices for security. Signup generates a secure 6-character code and stores user details; Login validates and redirects users to dashboard.php. A contact form logs queries into the database, and a history page enables users to track and manage past calculations. The UI includes a responsive design, navigation bar, social media links (Instagram, LinkedIn, X, GitHub), and a splash screen with animated logo. Educational content and modals (like “My Expertise”) offer guidance on budgeting and saving strategies. FinWise Academy’s mission is to make financial literacy accessible to everyone, encouraging informed financial decisions and long-term growth.

Technologies: HTML, CSS, JavaScript, PHP, MySQL

Ana/Dig Clock

Jun 2023

The Clock Project is a real-time, browser-based application designed to display the current system time using a visually appealing interface. Developed with core web technologies—HTML, CSS, and JavaScript—the project offers a practical implementation of time management and user interface interaction. The clock continuously updates every second to reflect the exact time, ensuring accuracy and responsiveness. The application can be presented in two formats: digital and analog. In the digital version, the time is shown numerically with hour, minute, and second components, offering 12-hour or 24-hour format toggle. The analog version mimics a traditional clock, using dynamically rotated hour, minute, and second hands built using CSS transformations and JavaScript. Additional features can include:

  • Live Date Display – showing the current day, month, and year.
  • Custom Themes – allowing users to switch between light and dark modes.
  • Alarm Functionality – enabling users to set specific time-based alerts with sound notifications.
  • Responsive Design – ensuring compatibility across desktops, tablets, and mobile devices.

From a technical standpoint, the project leverages setInterval() for continuous time updates, Date object methods to extract real-time values, and DOM manipulation to reflect changes in the UI. The design focuses on clean layout, readability, and intuitive controls. Overall, the Clock Project serves as a robust timekeeping tool while demonstrating effective use of JavaScript for real-time functionality and CSS for creative UI/UX design.

Technologies: HTML, CSS, JavaScript
